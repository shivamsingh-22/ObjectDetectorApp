# Object Detector App

An Android application that detects and identifies objects in real-time using the device camera and APIs.  
Capture images instantly and view detected object names with confidence scores.

---

## ✨ Features

✅ Capture Images using Device Camera  
✅ Real-Time Object Detection  
✅ API Integration for Image Processing  
✅ Displays Object Name & Confidence Score  
✅ Simple and Clean User Interface  

---

## 🚀 Tech Stack

**Technology:**  
- Android Studio  
- Kotlin  
- Google ML API
